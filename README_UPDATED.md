## What this site does

- Presents company information and branding
- Showcases services and menu options
- Displays a gallery of past events and dishes
- Provides an easy contact form for enquiries and bookings

## Main Pages

- Home
- About
- Services
- Gallery
- Contact

## Tech Stack

- Next.js (App router)
- React
- TypeScript
- PostCSS / global CSS
- Deployed on Vercel

## Local Development

Install dependencies and run the development server:

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000` in your browser to view the site.

## Build & Production

```bash
pnpm build
pnpm start
```

## Deployment

The site is deployed on Vercel. To update the live site, push changes to the repository connected to your Vercel project or use the Vercel CLI.
